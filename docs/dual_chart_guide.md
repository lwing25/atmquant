# 双图并排显示功能说明

## 功能概述

双图并排显示功能允许同时查看同一品种的两个不同周期图表，便于进行多周期分析和对比。

## 核心特性

### 1. 双图布局
- **左右分栏**：使用可调节的分割器，可以自由调整左右图表的显示比例
- **独立周期**：每个图表可以独立选择显示周期
- **默认配置**：左侧默认15分钟，右侧默认1小时

### 2. 时间轴同步
- **智能同步**：启用同步后，两个图表的时间轴会自动对齐
- **缩放联动**：在一个图表上缩放或拖拽，另一个图表会同步显示相应时间段
- **时间映射**：通过智能算法找到不同周期K线之间的时间对应关系

### 3. 周期选择
- **快速切换**：每个图表都有独立的周期选择面板
- **支持周期**：1分钟、5分钟、15分钟、1小时、日线
- **实时聚合**：切换周期时自动从基础数据聚合生成新周期K线

## 使用方法

### 在回测界面中使用

1. **打开K线图表**
   - 运行回测后，点击"K线图表"按钮
   - 图表默认以单图模式显示

2. **切换到双图模式**
   - 点击左上角的"双图模式"按钮
   - 界面会自动切换为左右两个图表

3. **调整周期**
   - 每个图表顶部都有周期选择面板
   - 点击对应的周期按钮即可切换

4. **启用同步**
   - 点击"启用同步"按钮
   - 两个图表的时间轴会自动对齐
   - 在任一图表上拖拽或缩放，另一个会同步

5. **调整比例**
   - 拖动中间的分割线可以调整左右图表的显示比例
   - 可以根据需要设置为1:1或其他比例

### 独立使用DualChartWidget

```python
from core.charts import DualChartWidget
from vnpy.trader.object import BarData

# 创建双图组件
dual_chart = DualChartWidget(
    left_period="15m",   # 左侧周期
    right_period="1h"    # 右侧周期
)

# 更新数据（使用1分钟K线）
dual_chart.update_history(minute_bars)

# 设置交易时段
dual_chart.set_trading_session_by_symbol("rb2505", "SHFE")

# 显示窗口
dual_chart.show()
```

## 组件架构

### DualChartWidget (双图容器)
- 管理两个EnhancedChartWidget实例
- 提供周期选择和模式切换界面
- 协调左右图表的交互

### ChartTimeAxisSync (时间轴同步管理器)
- 处理时间轴同步逻辑
- 建立不同周期K线的时间映射
- 实现缩放和拖拽的联动

### PeriodSelectorPanel (周期选择面板)
- 提供可视化的周期选择界面
- 支持快速切换常用周期
- 实时反馈当前选中周期

### 周期聚合

双图组件复用EnhancedChartWidget的聚合逻辑：

```python
def _aggregate_bars(self, minute_bars, target_period):
    # 使用EnhancedChartWidget的聚合方法
    return self.left_chart._aggregate_bars(minute_bars, target_period)
```

支持的聚合周期：
- 1m: 原始1分钟数据
- 5m: 5分钟聚合
- 15m: 15分钟聚合
- 1h: 1小时聚合（考虑交易时段）
- d: 日线聚合

## 使用场景

### 1. 多周期确认
- 左侧15分钟图观察短期趋势
- 右侧1小时图确认中期方向
- 同步时间轴对比同一时刻的表现

### 2. 入场点位优化
- 右侧日线图判断大趋势
- 左侧15分钟图寻找具体入场点
- 提高交易成功率

### 3. 策略验证
- 对比不同周期的信号强度
- 验证策略在多周期的一致性
- 优化策略参数

## 注意事项

1. **数据要求**
   - 建议使用1分钟K线作为基础数据
   - 数据量过大可能影响性能
   - 确保数据时间连续性

2. **同步限制**
   - 仅同步X轴（时间轴）
   - Y轴（价格轴）保持独立
   - 不同周期的Y轴缩放可能不同

3. **性能优化**
   - 时间映射建立后会缓存
   - 避免频繁切换周期
   - 大数据量时建议关闭部分指标
