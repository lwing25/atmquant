# 四图视图快速使用指南

## 什么是四图视图？

四图视图是一个2x2网格布局的高级图表功能，可以同时显示四个不同周期的K线图，帮助你全面把握市场节奏。

```
┌─────────┬─────────┐
│  5分钟  │ 15分钟  │  ← 短期趋势
├─────────┼─────────┤
│  1小时  │  日线   │  ← 中长期趋势
└─────────┴─────────┘
```

## 快速开始

### 1. 打开四图视图

运行策略回测后：
1. 点击"图表"按钮打开图表窗口
2. 点击顶部的"四图"按钮

就这么简单！四个不同周期的图表立即呈现。

### 2. 基本操作

**查看不同区域**
- 在任一图表拖动鼠标 → 所有图表自动同步
- 滚动鼠标滚轮 → 缩放所有图表

**调整布局**
- 拖动中间的分割线调整图表大小
- 让重要周期占据更大空间

**切换周期**
- 点击图表左侧的周期按钮
- 每个图表可独立切换周期

## 默认配置

| 位置 | 默认周期 | 适用场景 |
|------|---------|---------|
| 左上 | 5分钟   | 精确入场点 |
| 右上 | 15分钟  | 短期趋势 |
| 左下 | 1小时   | 中期趋势 |
| 右下 | 日线    | 大趋势判断 |

## 实战技巧

### 技巧1：多周期共振
1. 日线看大趋势（向上）
2. 1小时看回调结束
3. 15分钟看突破信号
4. 5分钟找精确入场

### 技巧2：关键位置确认
- 日线的支撑阻力最重要
- 在小周期找入场位置
- 多周期确认强度更高

### 技巧3：风险管理
- 大周期止损：日线级别
- 中周期止损：1小时级别
- 小周期止损：15分钟级别

## 视图切换

系统提供三种视图模式：

```
单图 ──→ 双图 ──→ 四图
 ↑        ↓        ↓
 └────────┴────────┘
```

- **单图**：专注单一周期，性能最好
- **双图**：对比两个周期，快速分析
- **四图**：全景视图，全面把握

根据分析需要随时切换！

## 常见问题

**Q: 四图会不会很卡？**
A: 不会。系统经过优化，只渲染可见区域。

**Q: 可以自定义周期吗？**
A: 可以。每个图表都可以独立切换周期。

**Q: 交易信号会显示吗？**
A: 会。所有四个图表都会显示交易信号和连线。

**Q: 能保存布局设置吗？**
A: 目前不能，这是未来计划的功能。
