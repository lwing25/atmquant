# 增强版K线图表组件使用指南

## 概述

`EnhancedChartWidget` 是基于vnpy的 `ChartWidget` 开发的增强版K线图表组件，提供了丰富的技术指标、交互功能和参数配置能力。

## 主要特性

### 🎯 技术指标支持

#### 主图技术指标
- **BOLL (布林带)**: 显示价格通道，支持周期和标准差配置
- **SMA (简单移动平均)**: 支持多周期显示，可自定义周期组合
- **EMA (指数移动平均)**: 更敏感的趋势跟踪，支持多周期
- **Smart Money (智能资金通道)**: 基于Pine Script的高级突破通道指标

#### 附图技术指标
- **Volume (成交量)**: 传统成交量柱状图
- **MACD**: 指数平滑异同移动平均线，包含MACD线、信号线和柱状图
- **RSI**: 相对强弱指标，范围0-100，包含超买超卖提示
- **DMI**: 趋向指标，包含+DI、-DI、ADX线

### 🖱️ 交互功能

#### 基础操作
- **鼠标滚轮缩放**: 向上缩小，向下放大
- **键盘导航**: 左右箭头键移动图表，上下箭头键缩放
- **双击放大**: 双击任意附图可放大/还原该附图

#### 高级拖拽功能
- **X轴扩展**: 拖拽图表右边缘可扩展X轴显示范围，超出数据范围
- **Y轴扩展**: 拖拽图表顶部/底部边缘可扩展Y轴显示范围
- **传统拖拽**: Ctrl/Cmd + 鼠标拖拽进行传统的平移操作

#### 显示控制
- **主图指标控制**: 通过复选框控制主图指标的显示/隐藏
- **附图指标控制**: 通过复选框控制附图的显示/隐藏
- **参数配置**: 每个指标都支持参数配置对话框

### ⚙️ 配置功能

每个技术指标都实现了 `IndicatorConfigurable` 接口，支持：
- 实时参数调整
- 参数验证
- 配置持久化
- 重置到默认值

## 使用方法

### 基本使用

```python
from core.charts import EnhancedChartWidget
from vnpy.trader.object import BarData

# 创建图表组件
chart = EnhancedChartWidget()

# 加载历史数据
chart.update_history(bar_data_list)

# 更新单个K线
chart.update_bar(new_bar)

# 显示图表
chart.show()
```

### 指标配置

```python
# 获取Smart Money指标实例
smart_money_item = chart.get_smart_money_item()

# 直接配置Smart Money指标
chart.configure_smart_money_channels()

# 获取指标当前配置
if smart_money_item:
    config = smart_money_item.get_current_config()
    print(f"当前配置: {config}")
```

### 内存管理
- 自动清理过期的绘图缓存
- 合理的数据结构设计，避免内存泄漏
- 支持数据的增量更新

## 扩展ViewBox功能

`ExtendableViewBox` 提供了增强的交互功能：

### X轴扩展
- 在数据右边缘拖拽，可以扩展X轴显示范围
- 支持超出数据范围的显示，便于预测和分析
- 自动限制最大扩展范围，避免过度扩展

### Y轴扩展
- 在图表顶部/底部区域拖拽，可以扩展Y轴范围
- 支持向上/向下扩展，最多300%的原始范围
- 双击顶部/底部区域可重置Y轴范围到自动适应

### 拖拽控制
- 智能检测拖拽区域，避免误操作
- 支持Mac的Command键和Windows的Ctrl键
- 保持与原有功能的兼容性

## 指标详细说明

### 布林带 (BOLL)
- **参数**: 周期(默认20)，标准差倍数(默认2.0)
- **组成**: 上轨、中轨(SMA)、下轨
- **用途**: 价格通道分析，超买超卖判断

### 移动平均线 (SMA/EMA)
- **SMA参数**: 周期列表(默认[5,10,20,60])
- **EMA参数**: 周期列表(默认[12,26,50])
- **用途**: 趋势跟踪，支撑阻力判断

### Smart Money通道
- **参数**: 
  - 嵌套通道: 允许多个通道同时存在
  - 强势收盘: 使用蜡烛体平均价而非收盘价
  - 归一化长度: 价格归一化的回看周期(推荐50-150)
  - 通道检测长度: 波动率分析周期(推荐10-20)
  - 量能分析: 显示右侧量能仪表盘
- **用途**: 机构资金进出判断，突破信号识别

### MACD
- **参数**: 快速周期(12)，慢速周期(26)，信号周期(9)
- **组成**: MACD线、信号线、柱状图
- **用途**: 趋势转换判断，买卖信号识别

### RSI
- **参数**: 周期(默认14)
- **范围**: 0-100
- **用途**: 超买超卖判断，背离分析

### DMI
- **参数**: 周期(默认14)
- **组成**: +DI、-DI、ADX
- **用途**: 趋势强度判断，方向性分析

## 最佳实践

### 指标组合建议
1. **趋势分析**: EMA + MACD + DMI
2. **震荡分析**: BOLL + RSI
3. **机构资金**: Smart Money + Volume
4. **综合分析**: BOLL + EMA + MACD + RSI

### 参数设置建议
1. **短线交易**: 使用较短周期参数
2. **中长线**: 使用较长周期参数
3. **不同品种**: 根据波动性调整参数
4. **回测验证**: 通过历史数据验证参数有效性

### 性能优化建议
1. **数据量控制**: 避免一次加载过多历史数据
2. **指标选择**: 根据需要选择必要的指标，避免过多指标影响性能
3. **参数调整**: 避免频繁修改参数，影响计算性能
4. **定期清理**: 定期清理图表缓存，释放内存

## 故障排除

### 常见问题

#### 指标不显示
- 检查复选框是否选中
- 确认数据量是否足够计算指标
- 检查指标参数是否合理

#### 性能问题
- 减少同时显示的指标数量
- 调整数据加载量
- 检查系统资源使用情况

#### 配置无效
- 确认参数在有效范围内
- 检查是否点击了"应用"按钮
- 重启图表重新加载配置

### 调试方法
1. 启用日志输出查看详细信息
2. 使用try-catch捕获异常
3. 检查数据完整性
4. 验证talib库是否正确安装
