# 多周期买卖点连线功能测试指南

## 快速测试

### 1. 运行回测

```bash
# 启动VN Trader
# 打开CTA策略回测模块
# 选择任意策略进行回测
```

### 2. 查看K线图表

回测完成后：
1. 点击"K线图表"按钮
2. 图表会显示1分钟K线和交易连线

### 3. 测试周期切换

在K线图表窗口：
1. 找到左侧的周期切换面板（竖排按钮）
2. 依次点击：1分钟 → 5分钟 → 15分钟 → 1小时 → 日线
3. 观察交易连线是否正确重绘到新周期的K线上

### 预期结果

✅ **1分钟周期**：
- 交易连线精确对应交易时间的K线
- 买卖点标记清晰可见

✅ **5分钟周期**：
- 交易连线智能匹配到包含交易时间的5分钟K线
- 例如：09:32的交易 → 显示在09:30的5分钟K线上

✅ **15分钟周期**：
- 交易连线智能匹配到包含交易时间的15分钟K线
- 例如：09:32的交易 → 显示在09:30的15分钟K线上

✅ **1小时周期**：
- 交易连线智能匹配到包含交易时间的小时K线
- 例如：09:32的交易 → 显示在09:00的小时K线上

✅ **日线周期**：
- 交易连线智能匹配到包含交易时间的日K线
- 所有当日交易都显示在当日K线上

## 验证要点

### 1. 智能匹配准确性

检查点：
- [ ] 交易时间早于K线时间时，连线显示在正确的K线上
- [ ] 交易时间晚于K线时间时，连线显示在正确的K线上
- [ ] 跨周期切换时，同一笔交易的连线始终显示在合理的位置

### 2. 自动重绘功能

检查点：
- [ ] 切换周期后，旧的交易连线被清除
- [ ] 新的交易连线正确绘制在新周期的K线上
- [ ] 连线颜色、样式保持一致（盈利红色、亏损绿色）

### 3. 视觉效果

检查点：
- [ ] 买入点标记（多头黄色下三角、空头品红色上三角）
- [ ] 卖出点标记（多头黄色上三角、空头品红色下三角）
- [ ] 成交量文本正确显示
- [ ] 连线为虚线样式

### 4. 边界情况

检查点：
- [ ] 交易数据为空时不崩溃
- [ ] K线数据量很少时正常显示
- [ ] 大量交易时性能可接受

## 已知限制

1. **交易时间超出K线范围**：
   - 如果交易时间早于第一根K线或晚于最后一根K线
   - 连线可能不显示（被`continue`跳过）

2. **极端周期切换**：
   - 从1分钟切换到日线时
   - 如果只有1-2天的数据，所有交易会聚集在1-2根K线上
   - 这是正常现象

3. **性能考虑**：
   - 如果交易数量 > 10000笔
   - 建议使用更大的时间周期查看
